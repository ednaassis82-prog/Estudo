Texto marcado

Podemos criar também um texto marcado usado a tag <MARK >>

Texto pequeno

Podemos também colocar textos reduzidos usando a tag < SMALL & GT;

Texto deletado

Texto inserido

Um texto sobrescrito permite escrever coisas como X2 usando <SUP >

Texto subscrito
