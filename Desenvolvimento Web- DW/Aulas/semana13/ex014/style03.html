CSS Externo

Um CSS externo, como o próprio nome sugere, está fora do documento HTML.

As principais vantagem no uso desse método são :

  • .Como o arquivo está "externo", fora do arquivo html, o seu conteúdo fica separado do estilo, e isso deixa tudo mais organizado.
  • .Ainda podemos permanecer os diversos estilo em cascata. Ainda sendo possível aplicar as duas outras estruturas.
  • >Um mesmo arquivo de estilo pode ser utilizado em vários páginas do seu sistema/site. Vamos aplicar também os exercícios.
